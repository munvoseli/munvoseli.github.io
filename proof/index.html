so there's been this math problem stuck in my head for years and i finally came up with a solution

Suppose that you have a rectangle with integer side lengths which you want to fill with 1 by integer rectangles along the gridboi. Which integer by integer rectangles can be filled by 1 by integer rectangles, without the 1 by integer rectangles forming any other rectangle?

It is only the 2x2 rectangle which cannot be filled.

proof

let us call the minimum side length b, and the maximum side length a.

case: b geq 3

there is a cool pattern for this which i must demonstrate with images.

case: b = 2, a > 2

there is a simple pattern for this. you can put 1x1 rectangles on opposite corners, and then fill the rest in with two rectangles.

case: b = 1

trivial

case: b = 2, a = 2

no

end of cases

in each case but 2x2, it was possible. 2x2 is the only rectangle in which it is impossible
